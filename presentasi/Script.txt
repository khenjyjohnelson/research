Introducing Omnitags

Omnitags bukanlah framework, melainkan mindset

Mindset bahwa semuanya bisa jauh lebih efisien jika kita mengejarnya (encapsulation)

Omnitags dapat membuat kita mengontrol waktu development

Tidak perlu menghafal nama tabel maupun field tabel

Input sekali, semua langsung kena

Less HARD CODING, more SOFT CODING

Tidak ada kata takut ketika mencari bug atau modifikasi kode

Semua variabel dapat dianalisa dengan mudah

Omnitags membuat kita lebih sedikit berpikir tentang hal-hal kecil (nama variabel, functions, method, dll) dan membuat kita fokus pada hal yang penting (algoritma)

Membantu Software Development Life Cycle menjadi lebih AGILE



 








Key points dari Omnitags
1. Encapsulation is key
2. Urutan tabel dan field dapat dimodifikasi
3. Isi dari variabel dapat diubah sesuka hati asal masih sama dengan nama tabel dan field di database
4. Kata-kata kunci tersebut dapat dihide dengan routes
5. Performance website tidak berpengaruh (malah lebih cepat)
6. Less hard code, more soft code
7. Programming jadi lebih AGILE
8. Input sekali, semua variabel langsung kena
9. key point lainnya yang mungkin belum saya sampaikan

Target yang pernah diraih
1. Aplikasi CRUD selesai dalam waktu singkat
	Rata-rata inisiasi sampai halaman beranda dan admin adalah 30 menit (rata-rata programmer lainnya bisa sampai 1 jam lebih meskipun menggunakan projek lama copy ke jadi projek baru)

2. Programming tidak terlalu melelahkan lagi untuk dimulai
3. Tidak menonton tutorial Youtube tentang cara implementasi Teknik ini selama 2-3 bulan terakhir (karena saya juga yang menemukan), hanya menggunakan chatGPT saja untuk membantu yang diperlukan saja














Kelebihan Omnitags
1. Inisiasi mudah dan tidak ribet (tidak perlu tahu cmd)
2. Waktu development jadi singkat
3. kelebihan-kelebihan lainnya

Kekurangan
1. Tidak boleh merubah nama file json (app.postman_environment.json). Jika ingin merubah nama file JSON tersebut, maka perlu merubah kodingan yang memanggilnya juga. Jika tidak dilakukan, website bisa loading cukup lama hingga memunculkan error.
